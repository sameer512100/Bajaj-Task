# Frontend - BFHL Hierarchy Visualizer

React + Vite frontend for submitting edge data to the BFHL backend and visualizing hierarchy/cycle output.

## Scripts

- npm run dev
- npm run build
- npm run preview

## Environment

Create a .env file when connecting to deployed backend:

VITE_API_BASE_URL=https://bajaj-backend-moem.onrender.com

If not set, the app falls back to the placeholder URL in src/lib/api.ts.
